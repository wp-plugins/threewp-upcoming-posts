== Description ==

Display upcoming posts in a widget.

Choose the category, number of posts and display format.

The display format can be configured. Leaving it blank will set it to the default value containing all keywords. The following keywords are available:

* %POST_TITLE% The post's title
* %POST_LINK% The href link to the post
* %POST_DATE% The post's scheduled date
* %POST_TIME% The post's scheduled time
 
To completely uninstall, unselect the "Enable linking", save the widget and then delete it.

== Installation ==

1. Unzip and copy the zip contents (including directory) into the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Add the widget to a sidebar.
